# Booking Rooms System

A modern full-stack Room Booking and Reservation System developed with frontend and backend integration for managing room reservations, user authentication, and booking workflows efficiently.

---

## 🚀 Features

- User Registration & Login Authentication
- Protected Routes & Authorization
- Room Listing & Details
- Room Availability Management
- Booking & Reservation System
- Occupied Dates Display
- Responsive Modern UI
- Admin & User Functionalities
- Dynamic Frontend Components
- API Integration
- Full Booking Workflow Management

---

## 🛠️ Technologies Used

### Frontend
- React.js
- JavaScript (ES6)
- Vite
- HTML5
- CSS3

### Backend
- Django/ restframework
- REST APIs

### Database
- MongoDB
